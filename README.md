lspreader
=========

A set of readers for the plasma code LSP's outputs.

Prerequisites
-------------
* `numpy>=1.10.1`
* `scipy>=0.16.0`
* `docopt` -- for scripts.

Install
-------

Using `pip`, and installing locally,

    git pull https://github.com/noobermin/lspreader.git
    cd lspreader
    pip install . --user
